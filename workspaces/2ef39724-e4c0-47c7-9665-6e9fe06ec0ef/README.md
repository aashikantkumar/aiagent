# Modern Calculator Web Application

A fully responsive and modern Calculator Web Application using HTML, CSS, and Vanilla JavaScript.

## Features

* Basic arithmetic operations (addition, subtraction, multiplication, division)
* Clear and backspace buttons
* Equals button to calculate the result
* Number buttons (0-9)

## Usage

1. Open the application in a web browser.
2. Click on the number buttons to enter a number.
3. Click on an operator button (+, -, \*, /) to select an operation.
4. Click on the equals button (=) to calculate the result.
5. Click on the clear button (C) to clear the display.
6. Click on the backspace button (DEL) to delete the last digit.

## Installation

1. Clone the repository using Git.
2. Open the index.html file in a web browser.
