# Project: react-todo-app

## Description
A simple To-Do application built with React and Vite.

## Tasks Checklist

- [x] **CREATE** `src/components/TodoForm.jsx`
  *Description:* Create the `TodoForm` component. This component will provide an input field and a button to add new todo items. It will manage its own local state for the input text using `useState`. On form submission, it will prevent the default behavior, call an `addTodo` function passed via props with the current input value, and then clear the input field. The component will export itself as default.

- [ ] **CREATE** `src/components/TodoItem.jsx`
  *Description:* Create the `TodoItem` component. This component will represent a single todo item in the list. It will receive a `todo` object (containing `id`, `text`, `completed`), a `toggleTodo` function, and a `deleteTodo` function as props. It will render the todo's text, a checkbox whose checked state reflects `todo.completed` and triggers `toggleTodo` on change, and a delete button that triggers `deleteTodo` on click. The todo text will have a conditional CSS class (e.g., `completed`) to apply a line-through style when `todo.completed` is true. The component will export itself as default.

- [ ] **CREATE** `src/components/TodoList.jsx`
  *Description:* Create the `TodoList` component. This component will be responsible for rendering the collection of `TodoItem`s. It will receive the `todos` array, `toggleTodo` function, and `deleteTodo` function as props. It will map over the `todos` array, rendering a `TodoItem` for each todo object and passing the individual `todo` data, `toggleTodo`, and `deleteTodo` functions as props to each `TodoItem`. The component will export itself as default.
