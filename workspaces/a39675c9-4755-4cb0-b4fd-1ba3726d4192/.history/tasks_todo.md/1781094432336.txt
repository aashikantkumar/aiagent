- [ ] **CREATE** `src/data/products.js`
  *Description:* Create a file to store product data as an array of objects. Each object should contain properties like id, title, description, image, price, and stock.

- [x] **MODIFY** `src/App.css`
  *Description:* Add styles for the product grid, cart sidebar, and other UI elements to ensure a consistent and responsive design.
